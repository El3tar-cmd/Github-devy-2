Quick Help

Workspace Simulation Triggers

Use the main Chat box to trigger complex automated agent processes offline:

  • Type "create dashboard page": Spawns the ReAct trajectory tracing loop, dynamically adds a Dashboard component to the file explorer, replaces editor code, checks off task lists, and mounts it inside the Browser Preview.
  • Type "diagnose script error": Launches the terminal autopilot compiler sweep, steps through diagnostic sweeper items, executes file correction patches, and updates console outputs.

AST Graph Inspector

Go to the **AST Graph** tab and tap on code nodes to inspect their imports and symbol tree, or double-click to load them in the code editor workspace.

API Mock Sandbox

Explore the **Sandbox Mocks** tab. Toggle Stripe and Auth0 triggers in the Webhook Simulator and tap "Dispatch Event" to watch the mock logs populate dynamically.
